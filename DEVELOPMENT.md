# Publish version

- lint -> `npm run lint`
- test -> `npm run test`
- build -> `npm run build`
- generate `CHANGELOG.md` -> `create-changelog`
- increase version in `package.json`
- commit -> `git commit -m "feat: ..., fix: ..., chore: ..."`
- tag -> `git tag v?.?.?`
- push -> `git push` and `git push --tags`
- create release on github -> ???
- publish `npm publish`
